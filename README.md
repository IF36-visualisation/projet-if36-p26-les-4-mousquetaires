# **Introduction**

## Données

Le [dataset](https://www.kaggle.com/datasets/nelgiriyewithana/billionaires-statistics-dataset/) utilisé par notre groupe contient des données sur les milliardaires à travers le monde contenant 2540 observations et un total de 35 variables qui seront plus ou moins utilisées.

Ce dataset provient de [Nidula Elgiriyewithana](https://www.linkedin.com/in/nidula/), un ingénieur chercheur en IA et regroupe tout un tas d'informations sur les milliardaires comme :

- Leur classement (rank)
- Le montant de leur fortune (finalWorth)
- Les industries dans lequel ces milliardaires opèrent (category, industries)
- Des informations personnels sur ces milliardaires (personName, age, country,state, city, countryOfCitizenship, gender, birthdate)
- La source de leur fortune (source)
- Le nom de l'organisme ou entreprise affiliée à ces milliardaires (organization)
- Un indicateur si ces milliardaires ont produit ou hérité de leur fortune (selfMade, status)
- Un indicateur de l'inflation du pays dans lesquels ces millionnaires vivent (cpi_country, cpi_change_country)
- Un indicateur du PIB des pays dans lesquels ces millionnaires vivent (gdp_country)
- Des informations sur l'éducation des pays dans lesquels ces millionnaires vivent (gross_tertiary_education_enrollment, gross_primary_education_enrollment_country)
- D'autres informations sur ces pays comme : L'espérance de vie, les taxes, la population (life_expectancy_country, tax_revenue_country_country,total_tax_rate_country, population_country)

Ce dataset est composées, pour les variables pertinentes, de données : discrètes, continues et nominale.

Nous avons choisi ce dataset car :

- Il est complet et toutes les informations nécessaires pour répondre à nos questions y figurent.
- L'idée de pouvoir en apprendre plus sur les milliardaires comme la provenance de leur richesse et leur répartition dans le monde nous intéresse.
- Avec la large quantitée d'informations variées, il est possible de chercher des réponses plus ou moins pertinentes sur des questions qu'on pourrait tous se poser. (proportions d'hommes/femmes, gagnée par l'héritage ou le travail, etc.)
